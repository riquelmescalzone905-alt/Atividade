# Sistema de Cadastro de Eventos da Cidade

Este projeto foi desenvolvido em Java como atividade acadêmica para praticar
programação orientada a objetos.

O sistema funciona no console e permite:
- Cadastro de usuário
- Cadastro de eventos na cidade
- Listar eventos
- Confirmar participação em eventos
- Cancelar participação
- Ver eventos em andamento, futuros e passados

Os eventos ficam salvos em um arquivo chamado events.data.

Estou no início dos estudos em Java, então o projeto é simples e pode ser melhorado.
